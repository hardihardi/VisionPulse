# **App Name**: VisionPulse Dashboard

## Core Features:

- Video Feed Upload: Upload video streams for real-time analysis via URL.
- License Plate Recognition: Detect and recognize license plates from the video stream in real-time, as a tool.
- PCU Calculation: Calculate Passenger Car Units (PCU) based on recognized vehicle types, in real-time.
- Moving Average Calculation: Compute and display 15-minute and 1-hour moving averages for license plate counts and PCU values.
- Interactive Charts: Generate interactive charts to visualize trends in license plate counts and PCU values over time.
- Real-time Data Display: Display real-time counts of license plates and PCU values for different vehicle types in an easy to read dashboard.
- Clean Code: The code must be clean and easy to understand

## Style Guidelines:

- Color scheme anchored on traffic monitoring, implying technology. Light primary color: Subtle sky blue (#B0E2FF).
- Background color: Very light cyan (#E0FFFF).
- Accent color: Grass green (#7CFC00).
- Body and headline font: 'Inter', a sans-serif font that would give the UI a clean, neutral feel.
- Use flat, vector-based icons for a modern, clean look.
- Implement a responsive grid layout for optimal viewing on both desktop and mobile devices.
- Use subtle animations to transition between data displays, for improved UX.
